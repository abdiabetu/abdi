# Multi-Tenant SaaS Dashboard

A premium, full-stack multi-tenant SaaS dashboard template built using React, Node.js, Express, and MongoDB.

## Features

- **Multi-Tenancy**: Complete data isolation using tenant reference keys (`tenantId`).
- **Role-Based Access Control**: Different roles (`Owner`, `Admin`, `Member`) with specific API authorization levels.
- **Dynamic Metrics**: SVG-based usage visualizations and KPI statistics.
- **Activity Auditing**: Detailed logging of user actions.
- **Stunning UI**: Custom dark theme styling with glassmorphism, responsive sidebar layout, and micro-animations.

## Directory Structure

- `backend/`: Express server, Mongoose models, authentication middleware, and API endpoints.
- `frontend/`: React app initialized with Vite using vanilla CSS variables for styling.
